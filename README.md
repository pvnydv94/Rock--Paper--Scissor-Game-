 ✊ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer and see who wins! The game features 
a clean interface, instant results, and automatic score tracking.

🚀 Features

* ✊ Choose **Rock**, **Paper**, or **Scissors**.
* 🤖 Play against the computer.
* 🎲 Random computer choice every round.
* 🏆 Instant winner announcement.
* 📊 Automatic score tracking.
* 🎨 Responsive and user-friendly interface.
* ⚡ Built using pure HTML, CSS, and JavaScript.

 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

▶️ How to Run

1. Clone the repository:

   
   git clone https://github.com/pvnydv94/Rock--Paper--Scissor-Game-.git
   
2. Open the project folder.
3. Open index.html in your web browser.

🎮 How to Play

* Click on **Rock**, **Paper**, or **Scissors**.
* The computer will randomly make its choice.
* The winner is decided using the classic rules:

  * ✊ Rock beats Scissors
  * ✋ Paper beats Rock
  * ✌️ Scissors beats Paper
* Scores are updated automatically after every round.
